# Financial-AI-OS (PFIOS)

Personal Financial Intelligence Operating System.

## Architecture (Arch-6)
- **apps/**: Web (Next.js) and API (FastAPI).
- **services/**: Business logic micro-units (Orchestrator, Risk Engine, etc).
- **skills/**: Atomic capabilities specs.
- **workflows/**: Periodic and event-driven automation.
- **policies/**: Human constitution and machine limits.
- **wiki/**: Obsidian knowledge layer.

## Quick Start
```bash
make init
make run
```
