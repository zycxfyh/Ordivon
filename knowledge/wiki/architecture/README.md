## Default Module Flow

For module-shaped work, the default flow is now:

1. read [AegisOS Design Doctrine](../../../docs/architecture/aegisos-design-doctrine.md)
2. classify the module as `Core / Pack / Adapter`
3. write the card with [Module Definition Template v2](../../../docs/product/module-definition-template-v2.md)
4. place the card under [module-cards](./module-cards/)
